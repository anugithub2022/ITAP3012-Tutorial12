In lecture 4, we discussed about the Client-Side Scripting, basics of javascript programming, Document Object Model and Events.

Topic details are as follows :

Introduction to JavaScript

JavaScript is an object-oriented, dynamically typed scripting language. It can be used to programmatically access and dynamically manipulate any aspect of the HTML documents.
Client-Side Scripting

It refers to the client machine (i.e., the browser) running code locally rather than relying on the server to execute code and return the result.
A client machine downloads and executes JavaScript code.
Programming basics of javascript

Briefing about arrays, loops, variables, objects, callback functions etc.
The Document Object Model(DOM)

When the web page is loaded, the browser creates a Document Object Model. The DOM document object is the root JavaScript object representing the entire HTML document.
Events and its types

JavaScript event is an action that can be detected by JavaScript. Discussion on different event types -
Mouse, keyboard and form events.
