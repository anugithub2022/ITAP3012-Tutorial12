In lecture 2, we discussed about history of HTML, HTML syntax and semantic. We also discussed about structure of HTML document,
HTML basic elements, web media.

Topic details are as follows :

HTML definition

HTML is defined as a markup language.
This “information about content” in HTML is implemented via tags (or more formally, HTML elements, but more on that later).
History of HTML

Initial implementation of HTML and HTTP between 1990 and 1991 by Tim Berners-Lee and Robert Cailliau.
In mid 1990s “browser wars” motivated many new tags and features such as CSS and JavaScript, however quick development and interoperability between browsers became a major issue.
HTML syntax

HTML documents are composed of textual content and HTML elements.
HTML element encompasses : The element name within angle brackets (i.e., the tag)
, HTML attribute that provide more information about the element (i.e., name=value pair)
, The content within the tag

Web Media - Color Models

RGB Model allow you to describe any color using red, green and blue. CMYK Model uses cyan-magenta-yellow-key (or black) colors.
