Internet defintions and history

Tim Berners-Lee and Robert Caillau invented web
Internet Protocols

A protocol is a set of rules that computers use when they communicate.
These protocols have been implemented in every operating system and make fast web development possible.
Uniform Resource Locators

A Uniform Resource Locator is the address of a given unique resource on the Web.
Web Servers

A web server is nothing more than a computer that responds to HTTP requests. Real-world web servers are often more powerful than your own desktop computer.
